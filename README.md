# 🛍️ Classic Fashion Electricals

A modern and responsive product showcase website built using **React.js + Vite + Tailwind CSS**.  
This project is designed for clients who want to display their products (like Amazon listings) in a clean UI with direct redirection to purchase links.

---

## 📌 Features

- 🖥️ Responsive UI (Mobile + Desktop)
- 🛒 Product showcase with cards
- 🔗 Direct redirect to product (Amazon links)
- 🎨 Smooth animations using Framer Motion & AOS
- ⚡ Fast performance with Vite
- 🧭 React Router navigation

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM
- **Animations:** Framer Motion, AOS
- **Icons:** Lucide React

---

## 📂 Project Structure

```

src/
├── assets/        # Images & static files
├── components/    # Reusable components (Navbar, Footer, ProductCard)
├── data/          # Product data
├── pages/         # Pages (Home, etc.)
├── App.jsx        # Main App component
├── main.jsx       # Entry point

````

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/classic-fashion-electricals.git
cd classic-fashion-electricals
````

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run locally

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```
